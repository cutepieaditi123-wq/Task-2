# Personal Portfolio Website

## Features

- Responsive Design
- Modern UI with Smooth Animations
- Hero Section
- About Me Section
- Skills Section
- Projects Showcase
- Contact Section
- Social Media Links
- Download Resume Button
- Interactive Navigation

- HTML5
- CSS3

└── assets/

This project was created to showcase my skills, projects, and achievements as a web developer while practicing front-end development concepts.
